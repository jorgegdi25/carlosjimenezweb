# Optimización de Imágenes Obligatoria
- Siempre optimiza las imágenes antes de usarlas.
- En proyectos de Next.js, usa siempre el componente `next/image` (`<Image />`) para compresión automática y lazy loading.
- Si usas imágenes estáticas, asegúrate de que estén en formato WebP o AVIF y comprimidas.
- Evita subir archivos originales gigantes (megabytes) que consuman ancho de banda innecesario.
<!-- END:image-optimization-rule -->
